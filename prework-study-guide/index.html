HTML

  • Head Element contains information(metadata).
  • Helps browser render the page
  • Not visible when a user visits the page
  • Body Element contains the content that is shown to the user
  • Contains all elements we want the user to see/interact with MUST be placed in this element

GIT Section

  • Git Status: Checks what branch we are currently on
  • Git checkout -b branch-name: Creates a new branch and switches to it
  • Git clone(REPOSITORY URL): Clone the repository from Github to the project folder
  • Git fetch: Get latest changes from the origin but not merge
  • Git Pull: Get latest changes from origin and merges
  • Git Branch: Shows a list of all branches
  • Git Merge:Merge the current branch into the master(first checkout to the master branch
  • Git checkout: Switch to the provided branch
